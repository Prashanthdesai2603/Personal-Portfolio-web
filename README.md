# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML5 and CSS3.

## Features

- Clean and modern design
- Fully responsive layout
- Smooth scrolling navigation
- Mobile-friendly interface
- No JavaScript dependencies
- Google Fonts integration

## Sections

1. **Home**
   - Hero section with name and tagline
   - "Hire Me" call-to-action button

2. **About**
   - Profile image
   - Personal bio
   - Skills list with hover effects

3. **Projects**
   - Responsive grid layout
   - 6 project cards with hover animations
   - Project images, titles, and descriptions

4. **Contact**
   - Contact form with input validation
   - Name, email, and message fields
   - Submit button with hover effect

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Montserrat & Roboto)

## Color Scheme

- Primary Blue: `#1565c0`
- Light Blue: `#90caf9`
- Background: `#f5faff`
- White: `#fff`
- Text: `#222`

## How to Run

1. Clone or download this repository
2. Open the project folder in VS Code
3. Install "Live Server" extension in VS Code (optional)
4. Right-click on `index.html` and select "Open with Live Server"
   - Alternatively, double-click `index.html` to open in your default browser

## Customization

1. Replace placeholder images in the `images` folder with your own
2. Update personal information in `index.html`:
   - Name and tagline in the Home section
   - About section bio and skills
   - Project details and descriptions
   - Contact form settings (if needed)
